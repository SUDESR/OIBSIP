# OIBSIP Task 1
BMI Calculator using Python

📌 Project Overview This is a simple Body Mass Index (BMI) Calculator built in Python. It takes a user’s weight (in kg) and height (in meters) as input, calculates BMI, and classifies it based on WHO standards. This project was created as Task 1 for my Oasis Infobyte Virtual Internship.

🎯 Features 📥 User-friendly Input – Enter your weight & height directly

🧠 Automatic BMI Calculation – Formula: weight / height²

📊 WHO-based Classification – Underweight, Normal, Overweight, Obese

🛡 Error Handling – Handles invalid or negative inputs gracefully

🖥️ Technologies Used Python 3 🐍

Command Line Interface (CLI)

Functions & Conditionals in Python

Error Handling with try-except
